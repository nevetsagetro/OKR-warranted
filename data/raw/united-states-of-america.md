### United States of America

Best Messaging Channels by Use Case (Bird's Recommendations)

- Transactional SMS: Toll Free Long Code (Phone Number)
- Promotional SMS: Toll Free Long Code (Phone Number)
- Two-Way Conversations: SMS with a two-way Toll Free Long Code

Sender Types Supported

- Alphanumeric Sender ID (Business names like "Bird" or "Netflix" that send one-way messages): No
- US Phone Number: Yes (SMS and MMS)
- US Short Code: Yes (SMS and MMS)
- International Phone Number: No
- Generic Sender ID (Sender IDs that don't clearly identify your brand): No

Restrictions

- Operator Registration Required: Yes
- Generic URL Shortening: No (Only branded URL shortening like yourbrand.link is allowed)
- Quiet Hours/Do Not Disturb: Yes (Monday-Sunday 20:00-09:00)

Additional Notes

- Also supported: US 10DLC, Toll Free Numbers
- See country-specific page for more information about short codes

Opt-out Rules

- Transactional traffic: Opt-out required. Stop language should be included in all registration messages
- Promotional traffic: Opt-out required. Stop language must be included in all registration messages
