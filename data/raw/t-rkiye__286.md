## türkiye

| Key | Value |
| --- | --- |
| MCC | 286 |
| Dialing code | 90 |
| Number portability | Yes |
| Concatenated message | Standard. Concatenated messages supported. |
| Service restrictions | Sender registration is a required. Documentation is needed. Before you start sending any traffic towards Türkiye for the first time, contact your dedicated account manager or [Support](mailto:support@infobip.com) so they can set up a specific route for you. |
| Service provisioning | Setup depends on sender provisioning time (depending on the network). Usually takes up to 5 days. |
| Sender availability | - Numeric (VLN provided by Infobip) only. - Alpha senders for local clients only. - Two-way: Virtual Long Number (inbound traffic only for SMS). |
| Sender provisioning | The average sender registration process time depends solely on network providers. |
| Two-way | Virtual Long Number |
| Two-way provisioning | It usually takes 1-2 days. |
| Country regulations | Only direct brand owners can terminate traffic towards Türkiye. Reselling is forbidden in Türkiye. Local dedicated VLN must be used for termination. Promotional traffic is allowed only for local companies (opt ins must be provided through specific portal on the Turkish Telecommunication Regulator side). |
| Country restrictions | Gambling, lottery and erotic content is forbidden by Turkish law. Opt-in is required for each end user (does not apply to transactional traffic) and opt-out option is required for all messages. Transactional messages must not include promotional expressions. Marketing messages must include a word describing the content, at the beginning of each message (like "PROMO") and exact or abbreviated client name/trade mark within message's text or sender name. No birthday, anniversary or similar personal messages without previously obtained opt-in. |
| Country recommendations | Promo traffic cannot be sent towards Türkiye from international clients, only from local. Local customers must use specific portal provided by the Turkish regulator to provide consents from end users. |
