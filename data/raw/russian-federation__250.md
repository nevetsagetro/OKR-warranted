## russian-federation

| Key | Value |
| --- | --- |
| MCC | 250 |
| Dialing code | 7 |
| Number portability | Yes |
| Concatenated message | Standard. Concatenated messages supported. |
| Service restrictions | There is traffic differentiation based on origin (local and international). For sending local or international traffic, you need to register your sender. For registration of a local sender, a Trademark is required. There is a monthly fee for each registered sender (depends on networks). Before you start sending traffic towards Russia, contact [Support](mailto:support@infobip.com) or your dedicated account manager. |
| Service provisioning | 5-10 days to configure the default account setup. It may be more if it is a specific setup (depending on MNOs). |
| Sender availability | Alpha and Short Codes only are available. Generic senders, and VLNs are forbidden. |
| Sender provisioning | The average sender registration process time depends solely on network providers. |
| Two-way | Standard, zero rated, and premium. Clients can choose Short Codes only for outbound and inbound messages. VLN can be set only for inbound (MO) traffic. If you want a Short Code, you have to make a special request towards each MNO for Short Code from your account manager or [Support](mailto:support@infobip.com). Short Codes are the only type able to receive inbound messages in Russia. |
| Two-way provisioning | To use Short Codes, you need to have a local entity. Provisioning time can take up to two weeks. |
| Country regulations | International and local traffic are separated. Local traffic is then separated into transactional, service and promo types which have different prices. There's specific billing for local traffic, so contact [Support](mailto:support@infobip.com) or your dedicated account manager for more information. Sender registration is mandatory. |
| Country restrictions | SPAM is forbidden. Messages of offensive or defamatory nature or messages inciting ethnic, racial or religious hatred, political agitation are forbidden. Messages containing information regarding gambling, blackmail, distribution of malicious software are forbidden. Traffic with erotic content, content that might violate public security, undermine the foundations of the constitutional order, state security is forbidden. Opt-in is mandatory. Advertising of alcoholic drinks, tobacco, smoking accessories, weapons, betting, raising funds for shared construction, security and exchange transactions is forbidden. |
| Country recommendations | Sending from the list of forbidden senders listed by MNO is prohibited. Sending international traffic from local senders is forbidden. Using generic senders is forbidden. |
