### US Virgin Islands

Best Messaging Channels by Use Case (Bird's Recommendations)

- Transactional SMS: Long Code (Phone Number)
- Promotional SMS: Long Code (Phone Number)
- Two-Way Conversations: SMS with a two-way long code

Sender Types Supported

- Alphanumeric Sender ID (Business names like "Bird" or "Netflix" that send one-way messages): No
- US Virgin Islands Phone Number: No
- US Virgin Islands Short Code: Yes
- International Phone Number: No
- Generic Sender ID (Sender IDs that don't clearly identify your brand): No
- US 10DLC (US 10-digit long code): Yes
- US Toll Free Numbers: Yes

Restrictions

- Operator Registration Required: No
- Generic URL Shortening: Yes
- Quiet Hours/Do Not Disturb: No

Additional Notes : Same USA regulations apply; you can use 10DLC, Toll-Free Numbers, and Short Codes

Opt-out Rules : No specific opt-out regulations
